# Minimal Analytics Dashboard

Одностраничный минималистичный дашборд на чистом HTML/CSS/JS для презентации проектов и статистики.

## Demo

Живое превью: https://karhalio.github.io/min.dashboard/

## Features

- Фиксированный сайдбар с группами навигации и подпунктами
- Светлая и тёмная темы с переключателем (сохранение в localStorage)
- Адаптивная сетка: карточки статистики, активность, таблица, прогресс-бар
- Блок «Проекты» с карточками и hover‑оверлеями (GitHub / Preview)

## Tech stack

- HTML5
- CSS3 (CSS-переменные, grid, flexbox, адаптив)
- Vanilla JavaScript (toggle темы, состояние сайдбара, мобильное меню)
